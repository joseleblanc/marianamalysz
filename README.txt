# 💌 Projeto Reconciliação

![Status](https://img.shields.io/badge/Status-Em%20Constante%20Adapta%C3%A7%C3%A3o-heart?style=for-the-badge&color=c61c44)
![Tecnologias](https://img.shields.io/badge/Tech-HTML5%20|%20CSS3%20|%20JS-blueviolet?style=for-the-badge)

Este é um projeto web interativo e personalizado, desenvolvido para expressar sentimentos e reconstruir laços. Através de uma narrativa gamificada, o site guia a utilizadora por desafios de memória e momentos de reflexão, culminando numa mensagem especial de reconciliação.

---

## 🚀 Fluxo da Experiência

O projeto é composto por três etapas principais interligadas:

1.  **O Desafio (`index.html`):** Uma validação de segurança onde a utilizadora deve inserir uma data marcante (ex: o dia em que se conheceram) para provar a sua identidade. Possui um fundo de partículas animadas e dicas progressivas.
2.  **O Contrato de Paz (`jogo.html`):** Uma página interativa com um "presente" bloqueado. Para o libertar, é obrigatório marcar um checkbox prometendo "não brigar com o desenvolvedor". Inclui uma barra de carregamento estilizada e uma citação inspiradora.
3.  **A Revelação (`mensagem.html`):** A fase final protegida por uma cortina. Ao clicar em "Revelar", o site dispara confetes, inicia a trilha sonora e apresenta um vídeo do YouTube junto a uma carta aberta sincera.

---

## 🛠️ Tecnologias e Recursos

* **Linguagens:** HTML5, CSS3 e JavaScript Vanilla.
* **Efeitos Visuais:** * **Canvas API:** Sistema de partículas dinâmicas (estrelas/chuva) unificado em todas as páginas.
    * **CSS Animations:** Transições de cortinas, efeitos de "shake" em erros e botões pulsantes.
    * **Canvas-Confetti:** Explosão de partículas coloridas na revelação final.
* **Multimédia:** Suporte para áudio em loop e integração de vídeo (Iframe).

---

## ⚙️ Como Personalizar (Template)

Este projeto foi criado para ser **aberto e utilizável como template**. Podes adaptar os seguintes pontos:

* **Textos:** Altera as frases dentro das tags `<h1>`, `<p>` e `<div>` em cada ficheiro HTML.
* **Data de Segurança:** No ficheiro `index.html`, altera a constante `correctAnswers` para a data que desejares.
* **Média:**
    * Substitui `musica.mp3` e `audio.mp3` pelos teus ficheiros de som.
    * Altera o link do YouTube no `iframe` do ficheiro `mensagem.html`.
    * Adiciona a tua própria imagem de `presente.png`.
