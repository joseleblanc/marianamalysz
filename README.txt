# 🌐 Projeto Mariana Malysz

Este projeto foi desenvolvido em **HTML, CSS e JavaScript**, com foco em aprendizado, estudo e uso pessoal.  
Está em **constante adaptação e atualização**, podendo servir como **template** para quem deseja reutilizar a estrutura.  

⚠️ **Atenção**: Projeto sem fins lucrativos, criado exclusivamente para fins **educacionais e pessoais**.  

---

## 📌 Sobre
O site tem como objetivo principal servir como **experiência prática** em desenvolvimento front-end.  
Você pode utilizá-lo como base para seus próprios projetos, realizando modificações em frases, textos e estilos.  

---

## ✨ Funcionalidades
- Estrutura simples e adaptável.  
- Estilo visual moderno com **CSS responsivo**.  
- Interatividade com **JavaScript**.  
- Permite fácil personalização de textos, cores e imagens.  

---

## 🚀 Tecnologias Utilizadas
- **HTML5** → Estrutura do site  
- **CSS3** → Estilização responsiva  
- **JavaScript (ES6+)** → Interatividade e dinamismo
